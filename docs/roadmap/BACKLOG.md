# Backlog

> Unprioritized ideas pool. Promoted to milestones when the time fits.

## Promotion criteria

An item leaves the backlog when at least one is true:

- It unblocks a current milestone goal.
- It is a known compliance / security obligation.
- A user pain has surfaced 3+ times.
- It enables a near-term revenue moment.
